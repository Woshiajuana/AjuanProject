
## 三. 操作命令

> npm run serve => 启动一个本地服务

> npm run dev   => 本地打包

> npm run test  => 测试打包

> npm run build => 生产打包

> node cmd -c                       =>  复制lib目录
> node cmd --copy                   =>  复制lib目录

> node cmd -d                       =>  清除dist目录
> node cmd --delete                 =>  清除dist目录


### 1. 样式命名

> 小写英文
> 中划线 -
